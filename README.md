# Unicor Mirror

A another Magic Mirror build.

### TL;DR;

```
$ cp config.json.example config.json
$ NODE_ENV='production' yarn build
$ yarn start
```

### Configure

All config are stored in `config.json`. See example config in `config.json.example`.

### How to develop this

```
$ yarn dev-client
$ yarn dev-backend
```
